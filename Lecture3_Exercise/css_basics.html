This box is absolutely positioned.

Relative positioning moves an element in relation to its original position, keeping the space it would have occupied intact. Absolute positioning, on the other hand, removes the element from the normal document flow and positions it relative to its closest positioned ancestor or the viewport if no ancestor is positioned.
