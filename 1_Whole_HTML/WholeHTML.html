HTML Structure

HTML is structured with a series of elements that define the layout of a webpage. Here's a basic example:

<!DOCTYPE html>
<html>
    <head>
        <title>Title of the page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
    </body>
</html>

Semantic Tags in HTML

Semantic tags clearly describe the meaning of the content inside them, improving both readability and accessibility. Some common semantic tags include:

  • <article> – Defines an article.
  • <section> – Defines a section in a document.
  • <nav> – Defines a container for navigation links.
  • <aside> – Defines content aside from the main content.
  • <footer> – Defines the footer of a document.
